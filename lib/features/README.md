# Features Module Structure

- `chat/` — chatbot models, provider UI (`ChatScreen`)
- `home/` — главный экран и навигационные карточки
- `mood/` — дневник настроения и статистика
- `tips/` — советы и практики
- `help/` — экраны поддержки и контакты
- `settings/` — экран настроек и экспорта данных

Правила:
- Каждый модуль содержит подкаталог `screens/`
- Дополнительные слои (`models/`, `widgets/`, `services/`) располагаем внутри соответствующего модуля
- Общие зависимости держим в `lib/core`
